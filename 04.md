# プロパティ(Property)～性質、属性～

## button1のプロパティを観察する

- BackColor
  -背景色
- Enabled
  -ボタンの有効/無効
- Font
  -書体の変更
- ForeColor
  -文字色の変更
- Location
  -ウィンドウ左上からの座標。
  -プログラムからだと、Left,Top
- Size
  -大きさ
  -プログラムからだと、WidthとHeight
- Text
  -表示する文字列
- TextAlign
  -文字列の揃え方
- Visible
  -表示するか、しないか

他にも、BackgroundImage, FlatStyle, Image, ImageAlignなど、時間がある時にいじってみよう。

## プログラムでプロパティの値を設定する方法
### ボタンを押したら、ボタンに「押された」と表示する

```cs
　button1.Text="おされた";
```

### ボタンを押したら、ボタンのLeftプロパティに0を代入する

```cs
  button1.Left=0;
```

### ボタンを押したらボタンを消す

```cs
 button1.Visible=false;
```

## (name)プロパティ
### button1の(name)を変更した時に変わった場所は？
-プロパティウィンドウのコンボボックスの名前
-コード上の名前も変わる。

### (name)プロパティとは何か？
-コントロールの名前を変更するもの

### (name)プロパティの注意点：
別のコントロールに同じ名前を付けることはできない。
